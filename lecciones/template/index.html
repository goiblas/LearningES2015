Mejoras en la declaración de objetos.

ES2015 introduce muchas mejoras en la creación y manejo de objetos. entre ellas están las siguientes:

  • Se puede evitar la estructura 'clave': valor, cuando existe una variable que se llama exactamente como queremos que se llame el atributo del objeto.
  • Se pueden deconstruir objetos (como en el pattern matching de lenguajes funcionales)
  • Existe un atajo para los métodos: se puede evitar la asignación de una función anónima a un parámetro, sólo se pone el nombre del parátmetro y se continúa como si se estuviera declarando una función nombrada.
